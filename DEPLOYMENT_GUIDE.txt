PART 1: GOOGLE APPS SCRIPT SETUP
==================================

STEP 1: Create/Open Google Apps Script
---------------------------------------
1. Go to: https://script.google.com/
2. Click "New Project" (or open existing script)
3. Delete any existing code
4. Copy ALL code from: GOOGLE_APPS_SCRIPT_FIXED.js
5. Paste into the editor

STEP 2: Connect Script to Your Sheet
-------------------------------------
Option A: Script is in the same spreadsheet
- If you created script from Google Sheet (Extensions ‚Üí Apps Script)
- Skip to Step 3

Option B: Script is standalone
- Select function: "connectToSheet"
- Click "Run" ‚ñ∂Ô∏è
- Allow permissions when prompted
- Should show: "Connected to sheet!"

STEP 3: Test Connection
------------------------
1. Select function: "testConnection"
2. Click "Run" ‚ñ∂Ô∏è
3. Check result:
   ‚úÖ SUCCESS = Everything connected correctly
   ‚ùå ERROR = Fix the error message

STEP 4: Deploy as Web App (CRITICAL!)
--------------------------------------
1. Click "Deploy" ‚Üí "New deployment"
   (or "Manage deployments" ‚Üí "New deployment")

2. Click gear icon ‚öôÔ∏è ‚Üí Select "Web app"

3. Deployment settings:
   - Description: "Contact Form Handler" (optional)
   - Execute as: "Me" (your email)
   - Who has access: "Anyone" ‚ö†Ô∏è MUST BE "ANYONE"
   - Click "Deploy"

4. Authorize access:
   - Click "Authorize access"
   - Choose your Google account
   - Click "Advanced" ‚Üí "Go to [Project Name] (unsafe)"
   - Click "Allow"

5. Copy the Web App URL:
   - It looks like: https://script.google.com/macros/s/AKfyc.../exec
   - ‚ö†Ô∏è IMPORTANT: Copy this URL - you'll need it!

STEP 5: Update HTML with Script URL
-------------------------------------
1. Open: index.html
2. Find line 1311:
   const scriptURL = 'https://script.google.com/macros/s/...'
3. Replace the URL with your NEW Web App URL from Step 4
4. Save the file

STEP 6: Test Contact Form
--------------------------
1. Open your website (locally or deployed)
2. Fill out contact form
3. Submit
4. Check your Google Sheet:
   https://docs.google.com/spreadsheets/d/1bS0kBJrsN8k8EA6HqZKqNoe9gozXoXxtWzmMMQ-Yavs/edit

PART 2: NETLIFY DEPLOYMENT (For Testimonials)
==============================================

STEP 1: Prepare Netlify Functions
---------------------------------
1. Make sure you have these files:
   - netlify/functions/get-testimonials.js
   - netlify/functions/submit-testimonial.js
   - netlify.toml
   - package.json

STEP 2: Set Up Neon Database
-----------------------------
1. Go to: https://neon.tech/
2. Sign up/Login
3. Create a new project
4. Copy your connection string
5. Run the SQL from: database-schema.sql
   (Creates the testimonials table)

STEP 3: Configure Netlify Environment Variables
------------------------------------------------
1. Go to your Netlify site dashboard
2. Site settings ‚Üí Environment variables
3. Add these variables:
   - DATABASE_URL = (your Neon connection string)
   - Example: postgresql://user:password@host/database

STEP 4: Deploy to Netlify
--------------------------
Option A: Deploy via Netlify Dashboard
1. Go to: https://app.netlify.com/
2. Click "Add new site" ‚Üí "Deploy manually"
3. Drag and drop your project folder
4. Wait for deployment

Option B: Deploy via Git
1. Push your code to GitHub
2. Connect GitHub repo to Netlify
3. Netlify will auto-deploy

STEP 5: Verify Deployment
--------------------------
1. Visit your deployed site
2. Test testimonials section
3. Submit a testimonial
4. Check if it appears (after approval in database)

PART 4: TROUBLESHOOTING
========================

Contact Form Not Saving Data:
-----------------------------
1. Check Google Apps Script ‚Üí Executions tab
2. Look for error messages
3. Verify "Who has access" is set to "Anyone"
4. Check sheet column headers match exactly:
   Timestamp, Name, Email, Phone, Message
5. Run testConnection() function again

403 Forbidden Error:
--------------------
- Script not deployed correctly
- "Who has access" is "Only myself" (should be "Anyone")
- Fix: Redeploy with correct permissions

Testimonials Not Working:
--------------------------
- Netlify functions only work when deployed
- Check environment variables are set
- Verify Neon database connection
- Check Netlify function logs

CORS Errors:
------------
- Already handled with mode: 'no-cors'
- If still seeing errors, check script deployment
